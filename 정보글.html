git hub내가 만든 코드를 웹서버에 무료로 간단하게 연결할 수 있도록 도와주는 사이트

사용방법>>git bub에 로그인>>new repository(저장소)클릭>>
repository name 정하기>>Initialize this repository with a README 체크하기>>
밑에 create repository클릭>>up load클릭>>내가 만든 파일들 업로드>>commit changes 파일들 내용적기>>
파일들을 이용해서 내가 사이트를 운영할수있도록 무료로 컴퓨터와 웹서버를 제공해줌
>>settings클릭>>GitHub Pages란 에 source밑에 none을 master branch로 변경후 save>>
그러면GitHub Pages란에 나의 사이트 주소를 제공해줌
